**Internet Programming: EpicPanel – Comic Book E-Commerce Platform**

This project presents the design and implementation of a full-stack e-commerce web application for an independent online comic bookstore. Built on the Django framework with an SQLite backend, the system supports a multi-role user model (customer, staff, admin) and implements core e-commerce functionality including a filterable comic catalogue, user authentication, cart and checkout workflows, and a review system with aggregated ratings. The data model comprises eight relational entities (Users, ComicBook, Category, Publisher, CartItem, Order, OrderItem, Review) with normalized foreign-key relationships. The frontend is rendered via Django templates with Bootstrap for responsive layout. Future extensions include payment gateway integration (Stripe/PayPal), order tracking, and wishlist functionality.
